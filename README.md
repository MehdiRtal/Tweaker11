# WinCleanstall
Create an automated, clean and custom Windows installation

## Installation
* Clone the repo.
* Install requirements with the following command :
 ```
 pip install -r requirements.txt
 ```
* Run the script.

## Features
* Automatic Windows download/installation
* Post-installation tweaks
